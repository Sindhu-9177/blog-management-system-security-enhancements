# Blog Management System - Security Enhancements

## Overview

A secure Blog Management System developed using PHP, MySQL, Bootstrap 5, HTML, CSS, and JavaScript. The application allows users to create, manage, search, and organize blog posts through a responsive dashboard while implementing important security features.

## Features

* User Registration and Login Authentication
* Password Hashing for Secure Storage
* Session Management
* Create, Read, Update, and Delete (CRUD) Operations
* Search Functionality for Blog Posts
* Pagination for Better Navigation
* Responsive Dashboard using Bootstrap 5
* Form Validation
* Prepared Statements
* SQL Injection Protection
* User Roles and Permissions (Admin / Editor)
* Role-Based Access Control
* Created Date Tracking
* Git and GitHub Version Control

## Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* XAMPP
* Git
* GitHub

## Security Enhancements

* Prepared Statements for Database Queries
* Password Hashing using PHP Password API
* User Input Validation
* Session-Based Authentication
* Role-Based Access Control
